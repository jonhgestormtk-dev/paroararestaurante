# **App Name**: Paroara Marajoara

## Core Features:

- Live Digital Menu: A responsive product grid displaying Marajoara delicacies synced directly with a cloud database.
- Artisanal Category Filters: Horizontal scrollable navigation to filter items from Regionais to Bebidas with smooth active-state animations.
- Unified Shopping Cart: Persistent mobile-first tray for item selection, subtotal calculation, and customized order observations.
- WhatsApp Order Generator Tool: A smart tool that reasons through the cart data to format and export a professional order message directly to WhatsApp.
- AI Dish Storyteller: A tool powered by generative AI that creates poetic, cultural dish descriptions based on the specific Amazonian ingredients listed.
- Order Management Portal: A secure administrative dashboard to track orders, update dish availability, and change item pricing in real-time.
- Auth-Protected Admin Hub: Restricted access panel using modern authentication to ensure secure management of the restaurant settings and media banners.

## Style Guidelines:

- Primary color: Rich Soil Brown (#4B2E1F) for a grounding and premium rustic feeling.
- Background color: Deep Espresso Shadow (#1C1816), an earthy, desaturated version of the primary to create depth in a dark scheme.
- Accent color: Amazonian Caramel (#A87442), an analogous warm tone used for highlights and primary CTA contrast.
- Headline: 'Cinzel Decorative' for Marajoara heritage; Subhead: 'Cormorant Garamond' for elegant serif flair; Body: 'DM Sans' for readable geometric UI. Note: currently only Google Fonts are supported.
- Refined, minimal organic icons representating nature, craft fibers, and artisanal culinary tools.
- Wide spacing with breathable padding and fine 6px-radius cards inspired by handmade regional pottery.
- Graceful fade-in transitions and subtle hover lifts to evoke a quiet, sophisticated Amazonian atmosphere.
